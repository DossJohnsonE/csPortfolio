Dice!

Tap the screen and watch the dice roll change!

This project was a lot of mathematical work for me. The hardest part was placing each die right, along with the circles in each one. I ended up drawing a sketch of what I wanted the screen to look like on a piece of paper, and then I mapped it out and figured out the measurements from there. A way I reduced my code was by creating a lot of if statements for the drawing of the dots on the die. Instead of drawing every possible die separately, I mapped it out and looked to see which numbers each dots on each possible roll lined up. I noticed patterns for each dot, such as for the top left dot and the bottom right dot, it showed up whenever the roll was greater than or equal to 2. So, I made an if statement that if the roll of the die was greater than or equal to two, it would draw those two dots. This made my code much shorter and easier for me to follow and edit! This project is probably the one I'm most proud of due to my code!
